#  VS Code Shortcut Keys (Windows/Linux & Mac)

This documentation summarizes **Visual Studio Code shortcut keys** across **10 aspects of development workflow**, with **5 key shortcuts per aspect** — ideal for productivity, navigation, debugging, and collaboration.

---

## 1.  Navigation & File Management

| Action | Windows / Linux | macOS |
|--------|-----------------|-------|
| Open Command Palette | `Ctrl + Shift + P` | `Cmd + Shift + P` |
| Quick Open File | `Ctrl + P` | `Cmd + P` |
| Toggle Explorer Sidebar | `Ctrl + B` | `Cmd + B` |
| Go to File Definition | `F12` | `F12` |
| Go to Symbol in File | `Ctrl + Shift + O` | `Cmd + Shift + O` |

##  Workflow Example

**Scenario:** You’re fixing a bug in a Python project.

```plaintext
Ctrl + P                → Quick open the file
Ctrl + /                → Comment/uncomment debug lines
F5                      → Run debugger
Ctrl + Shift + G        → Open Git panel
Ctrl + Enter            → Commit fix
Ctrl + `                → Open terminal for tests
pytest -v               → Run unit tests
